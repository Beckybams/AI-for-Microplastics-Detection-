Overview

This project applies Artificial Intelligence (AI) and Machine Learning (ML) techniques to detect and classify microplastics in environmental samples. Using image-based or sensor-derived data (real or synthetic), the model identifies the presence, type, and distribution of microplastics to support environmental monitoring and research.
